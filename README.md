## Installation

```plaintext
git clone https://github.com/AymaneChaouki/react-router-HomeBudget-app.git
cd react-router-HomeBudget-app
npm run dev
```

## Tools used

-   Reactjs
    -   Context api
    -   useState, useEffect
    -   react router v6
-   LocalStorage
